# NenovCars

*A simple application developed for my exam in Software University in React course. 

## You can:
  *Login;
  *Register;
  *Browse latest ads;
  *Add to your favorites;
  *Create ad;
  *Edit your ad;
  *Delete your ad;
  *Search;
  
## Tech Stack:
  # Backend - 
    *ASP.Net Core 2.2;
    *SQLite;
    *Entity Framework Core;
  # Frontend -
    *React;
    *TypeScript;
    *mobX;
    *AntDesign;
    *Toastr;
